# E-Commerce Customer Churn Prediction
### Machine Learning Capstone | WGU B.S. Computer Science (C964)

## Business Problem
Customer churn is one of the most costly challenges in e-commerce. 
Identifying at-risk customers before they leave allows retention teams 
to intervene with targeted offers, loyalty incentives, or personalized 
outreach — directly protecting revenue. This project builds a 
data-driven churn prediction pipeline applicable to any e-commerce 
operation, inspired by real merchandising and customer analytics work 
at scale.

---

## What This Project Does
- Predicts whether an e-commerce customer will churn using supervised 
  machine learning
- Segments customers by behavior using unsupervised clustering 
  (K-Means) to identify distinct risk profiles
- Compares model performance across a Decision Tree classifier and 
  regression-based approaches
- Delivers actionable churn risk scores with feature importance 
  analysis to explain *why* customers are flagged

---

## Technical Approach

| Stage | Method |
|---|---|
| Data cleaning & EDA | Pandas, Matplotlib, Seaborn |
| Feature engineering | Scikit-learn preprocessing |
| Clustering | K-Means (unsupervised segmentation) |
| Classification | Decision Tree Classifier |
| Evaluation | Accuracy, precision, recall, confusion matrix |
